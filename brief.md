## Суть проекта
Персональный лендинг фрилансера — цифровые продукты под ключ.
Один HTML-файл без фреймворков и зависимостей. Тёмная тема, анимации, mobile-first.

---

## Стек
- Чистый HTML + CSS + JS — один файл `landing.html`
- SVG-иконки встроены в HTML
- Без CDN, без npm, без фреймворков
- Canvas-анимация (нейросеть → топография при скролле)
- Scroll-reveal через IntersectionObserver
- FAQ-аккордеон
