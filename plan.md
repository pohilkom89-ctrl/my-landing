  ```bash
  gh auth login
  cd C:\Users\Миша\Documents\my-project\my-landing
  gh repo create my-landing --public --source=. --remote=origin --push
  ```

- [ ] **Опубликовать лендинг онлайн**

## Идеи для развития

### Продуктовые идеи
- **Пакеты услуг** — Start / Pro / Premium с фиксированной ценой
- **Шаблоны ботов** — продавать готовые решения для разных ниш
- **Подписка на поддержку** — ежемесячное обслуживание бота/сайта

### Контентные идеи для канала
- "Что такое Telegram-бот и зачем он бизнесу"
- "Сколько стоит сделать сайт в 2025"
- "До/после: как бот заменил менеджера"
- Кейсы реальных проектов

### Технические идеи
- Добавить тёмный/светлый режим
- Сделать мультиязычную версию (RU/EN)
- Создать отдельные страницы под каждую услугу
